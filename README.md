# OOPWorkshop #1 — Time Class

## Description
This project implements a class called `Time` using Object-Oriented Programming in C#.  
It allows handling hours, minutes, seconds, and milliseconds, validating the input values and performing basic time operations.

## Project Structure

### Backend
Contains the main system logic:

- `Time` class
- Private attributes for hour, minute, second, and millisecond
- Properties to access the data
- Validation methods
- Conversion of time to total seconds
- Text representation of the time

### Frontend
Console application that allows:

- Creating objects of type `Time`
- Displaying time values
- Testing the implemented methods

## Main Features

- Management of hours, minutes, seconds, and milliseconds
- Validation of correct time values
- Conversion to total seconds
- Readable time format output

## Technologies Used

- Language: C#
- Platform: .NET
- IDE: Visual Studio
